This folder contains the data obtained of running the neural networks with the updated feature parameters TO OBTAIN THE BINARY CROSS ENTROPY. 
Folder called MM contains the results of training and testing with mixed density states, PP contains the following with pure.

Each folder contains then a set of two numbers which indicated the TW/TO negativity of the used Neural Network, being 0 the first number and Y the second one, which indicated the starting point of the interval so 00 means [0.0,0.1] interval, 01 [0.1,0.2] and so on.

Inside of each of these folders the files are called NNX_MM_0Y, being X the neural network is was used one, MM TW/TO Mixed (rest. PP TW/TO pure) and 0Y the interval as before.

The neural networks tested on are
NN1 - 16:8:1
NN2 - 9:8:1
NN3 - 9:6:1
NN4 - 9:4:1
NN5 - 9:2:1
